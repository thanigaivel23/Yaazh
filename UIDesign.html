Our UI/UX Design Process

Yaazh"s UI/UX design process involves user research, wireframing, prototyping, user testing, and iterative refinement to create intuitive and engaging user experiences that enhance usability and drive user satisfaction.

1. Research

On field research is a process of defining the scope before designing

2. Empathies

Emphathising is the only way to know the exactfeel of user while using the application

3. Create

This phase converts the prototype into design

4. Test

Test is a phase where a satisfaction of user request is rechecked

5. Develop

Development is a phase of converting design into working prototype
